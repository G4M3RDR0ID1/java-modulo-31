# Que tal praticar?
Você percebeu que dá pra fazer muita coisa com programação funcional utilizando a Stream API, não é? Streams resolvem problemas muito comuns de uma forma simples! Vamos exercitar um pouco.

* A partir de uma lista de dados do objeto Empresa, filtre todos os funcionários de qualquer empresa que tenham nascido entre 1975 e 1985 que possuem um salário menor ou igual a R$3000,00.

* Crie um método que receba uma lista de números e retorne a soma dos números pares contidos na lista.

* Crie um método que receba uma lista de empresas e retorne os 30 maiores sálarios de funcionários CLT que possuem menos de 25 anos.
